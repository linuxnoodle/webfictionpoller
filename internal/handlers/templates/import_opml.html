Import OPML

Upload an OPML file exported from your RSS reader (FreshRSS, Feedly, etc.). The importer will match feeds to supported providers and add them as tracked series.

Supported providers:

  • RoyalRoad
  • SpaceBattles
  • SufficientVelocity
  • QuestionableQuesting
  • FanFiction.net (requires FlareSolverr)

Restore Backup

Upload a JSON backup file previously exported from WebFiction Poller. This restores all series with ratings, status, chapters, and read state. Existing series will be updated; new series will be created.
